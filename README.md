# Day 9 – Design Patterns

## Deep Theory
Covered important JavaScript design patterns such as **Observer Pattern, Factory Pattern, Singleton, Module Pattern, Event Bus, and MVC Architecture**.  
These concepts help in building **scalable, reusable, and maintainable applications**.

## componentFactory.js
Implemented the **Factory Pattern** for dynamic component creation.  
The factory creates different UI components like **Button, Card, and Modal** based on the provided type.

## eventBus.js
Built an **Event Bus system** to handle communication between components using **subscribe, emit, and unsubscribe** methods.  
This helps in creating an **event-driven architecture**.
